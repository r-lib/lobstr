# Call stack tree

Shows the relationship between calls on the stack. This function
combines the results of
[`sys.calls()`](https://rdrr.io/r/base/sys.parent.html) and
[`sys.parents()`](https://rdrr.io/r/base/sys.parent.html) yielding a
display that shows how frames on the call stack are related.

## Usage

``` r
cst()
```

## Examples
